# mtg-tracker-client
How to use my MTG Tracker:

Get the Backend Sheet: Make a copy of this Google Sheet (contains the database and script): [LINK TO YOUR BLANK SHEET COPY]

Deploy the API:

In the Sheet, go to Extensions > Apps Script.

Click the blue Deploy button (top right) > New Deployment.

Select type: Web App.

Description: "MTG Tracker".

Execute as: Me.

Who has access: Anyone (This is crucial so the website can talk to the sheet).

Click Deploy and copy the Web App URL.

Connect:

Go to: https://wmmalwareresearch.github.io/mtg-tracker-client/

Paste your Web App URL into the box.
